docs_troubleshooting_title

docs_troubleshooting_lead

No credits

Refresh license or purchase Assessment / Validation Check.

Tenant not registered

Run registration again and verify API URL and token.

Report link does not open

Check link expiry and public base URL configuration.

Dashboard is empty

Wait for scan completion and refresh dashboard data.

Wrong language

Refresh license so tenant language can be updated.
